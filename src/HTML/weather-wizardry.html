Weather Wizardry : Maître du temps

Informations sur le projet

Cadre du projet : Code Game Jam 2023

Technologies utilisées : C# sur Unity

Rôle : Développeur Backend

Statut : Terminé

Description

Ma première participation à la Code Game Jam a été une expérience surprenante qui m'a permis de gagner le troisième prix.

La Code Game Jam (CJG) est une compétition qui regroupe plusieurs IUT et dont l'objectif est de créer un jeu en 2 jours basé sur un thème précisé juste avant le début de l'évènement. Le jury présent aide les groupes qui participent et jugent les projets une fois la CGJ terminée.

Avec 37 équipes de 6 personnes en confrontation et sur le thème "Maître du temps", cette Code Game Jam fut plus compliquée que prévu. Malgré les divers obstacles rencontrés, tels que trois coéquipiers absents et sept heures passées à résoudre des problèmes de .NET sur une machine, notre équipe n'a pas baissé les bras et a réussi à atteindre la troisième place. Nous avons beaucoup appris durant cet évènement, notamment sur le fonctionnement du logiciel Unity et de la programmation en C#.

Nous avons donc réalisé un platformer répondant aux critères requis, en se répartissant chacun les tâches et en s'entraidant mutuellement à 3 dans une équipe.

Contribution

Lors de cet évènement notre équipe s'est organisée parfaitement afin de palier aux problèmes rencontrés. Cela nous a permis de combler le manque de temps et de personnel. Mon rôle durant cette CGJ a été de mettre en place les éléments sur Unity, de créer les scripts de certains objets comme les personnages principaux ou encore les livres et les collisions.
