Portage des Aventuriers des Rails : Autour du monde

Informations sur le projet

Cadre du projet : SAÉ de développement de deuxième semestre de BUT

Technologies utilisées : Java 18, JavaFX 18

Rôle : Développeur Frontend et Backend

Statut : Terminé

Description

Ce projet était notre SAÉ de développement de deuxième semestre, comptant pour les composantes de Développement et d'Optimisation du BUT Informatique (UE 1 et 2). Nous devions faire un portage du jeu de plateau Les Aventuriers du Rail : Autour du monde avec le langage de programmation Java et le framework d'IHM JavaFX.

Le projet s'est déroulé en 3 étapes. La première étape était de développer le jeu suivant les règles et une base de code fournie par les enseignants au lancement de la SAÉ. Nous devions donc développer en Java les différentes mécaniques de jeu, et la boucle principale de jeu. Nous devions régulièrement faire des tests unitaires à l'aide de l'API Jupiter, et nous pouvions tester le jeu avec une interface web de tests fournie dans le projet.

La deuxième étape était de développer une API Graphes afin de l'intégrer au jeu pour pouvoir utiliser des algorithmes de calcul de routes les plus courtes reliant deux ou plusieurs villes. Cela permettrait donc à l'interface d'afficher au joueur les routes les plus courtes lui permettant de compléter une destination, mais aussi pour permettre à un ordinateur de pouvoir prendre des décisions stratégiques.

Enfin, la dernière étape du projet était de réaliser une interface graphique avec JavaFX. Pour ce faire, nous avons utilisé le SceneBuilder pour générer des fichiers FXML que nous avons directement intégré au projet, en connectant les composants entre eux. Pour la partie jeu, une correction nous a été fournie afin d'avoir à disposition les possibles fonctionnalités manquantes de la première partie.

Contribution

Dans ce projet, je me suis occupé de certaines mécaniques de jeu (piocher une carte, capturer une route, échange des pions...), de l'implémentation de l'algorithme de Dijkstra dans la classe Graphe, ainsi que de la liaison entre les composants internes et l'interface graphique.
