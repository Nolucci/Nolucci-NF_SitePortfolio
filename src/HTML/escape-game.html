Création d'une présentation d'Escape Game

Informations sur le projet

Cadre du projet : Première SAE de premier semestre en BUT

Technologies utilisées : HTML/CSS

Rôle : Développeur Frontend et Backend

Statut : Terminé

Description

Ce projet était notre SAÉ de développement de premier semestre. Nous devions faire un site de présentation du célèbre jeu : Escape Game.

Placé en correspondance avec une équipe d'une autre classe, ce projet s'est déroulé en 3 étapes. La première étape était de présenter une idée de thème à donner à l'autre équipe. La deuxième était de récupérer le thème de l'autre équipe et de produire un site de présentation d'un escape game sur le thème qui nous a été attribué. La création des salles et des énigmes sous HTML/CSS devait rentrer dans le contexte et être pertinent. La dernière étape consistait enfin à noter la production de l'autre équipe.

Lors de cette SAE nous avons du communiquer avec l'équipe "client" pour savoir ce qu'il fallait améliorer au fur et à mesure de l'avancée du projet. Une vidéo montrant une réunion entre les deux équipes a été demandée afin de confirmer que les deux équipes communiquaient.

Contribution

Dans ce projet, je me suis occupé de l'aspect graphique du site notemment sur les parties présentant les différentes salles. Cela m'a permis d'acquérir des compétences en HTML et CSS en début de première année de BUT.
