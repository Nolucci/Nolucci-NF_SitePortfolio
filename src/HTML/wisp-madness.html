Wisp Madness : Légende éthérée

Informations sur le projet

Cadre du projet : Code Game Jam 2024

Technologies utilisées : C# sur Godot

Rôle : Développeur Backend

Statut : Terminé

Description

Ce jeu est l'un de mes projets les plus aboutis pour lequel mon équipe a gagné le deuxième prix de la Code Game Jam.

La Code Game Jam (CJG) est une compétition qui regroupe plusieurs IUT et dont l'objectif est de créer un jeu en 2 jours basé sur un thème précisé juste avant le début de l'évènement. Le jury présent aide les groupes qui participent et jugent les projets une fois la CGJ terminée.

Avec 43 équipes de 6 personnes en confrontation et sur le thème "Légende Ethérée", la Code Game Jam a été très productive pour mon groupe qui a réussi a créer un jeu intéressant et addictif apprécié du jury. Nous avons beaucoup appris durant cet évènement, notamment sur le fonctionnement du logiciel Godot et de la programmation en C#.

Nous avons donc réalisé un survival game répondant aux critères requis, en se répartissant chacun les tâches et en s'entraidant mutuellement.

Contribution

Lors de cet évènement notre équipe, ayant choisi une multitude de défis, s'est organisée en petits groupes pour tout développer. J'ai organisé le groupe de 6 personnes en équipes de 3 en attribuant des tâches à chacune. Mon rôle durant cette CGJ a été de mettre en place les éléments sur Godot, de créer la map et les scripts de certains objets comme les ennemis ou le personnage principal.
